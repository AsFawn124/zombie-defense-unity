# 占位精灵图说明

此文件夹用于存放游戏精灵图资源。

## 自动生成占位资源

在Unity编辑器中，使用菜单 `Tools → Create Placeholder Assets` 可以自动生成以下占位精灵：

### 防御塔
- `Tower_Base.png` - 基础炮台（蓝色圆形）

### 敌人
- `Enemy_Normal.png` - 普通僵尸（红色圆形）
- `Enemy_Fast.png` - 快速僵尸（黄色小圆形）
- `Enemy_Tank.png` - 坦克僵尸（紫色大圆形）
- `Enemy_Boss.png` - BOSS（红色大圆形）

### 其他
- `Base.png` - 基地（绿色方形）
- `Bullet.png` - 子弹（黄色小圆形）
- `Range_Indicator.png` - 范围指示器（黄色圆环）

## 替换为正式美术资源

1. 将正式美术资源放入此文件夹
2. 保持文件名与上述一致
3. 在Unity中选中图片，设置Texture Type为Sprite (2D and UI)
4. 点击Apply应用设置
