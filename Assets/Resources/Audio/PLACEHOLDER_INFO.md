# 音频资源说明

此文件夹用于存放游戏音频资源。

## BGM (背景音乐)

| 文件名 | 用途 | 建议时长 |
|--------|------|---------|
| `bgm_main.mp3` | 主界面BGM | 60-120秒 |
| `bgm_battle.mp3` | 战斗BGM | 60-120秒 |
| `bgm_boss.mp3` | BOSS战BGM | 60-120秒 |
| `bgm_victory.mp3` | 胜利BGM | 5-10秒 |
| `bgm_defeat.mp3` | 失败BGM | 5-10秒 |

## SFX (音效)

| 文件名 | 用途 |
|--------|------|
| `sfx_shoot.wav` | 射击音效 |
| `sfx_hit.wav` | 受击音效 |
| `sfx_explosion.wav` | 爆炸音效 |
| `sfx_button.wav` | 按钮点击 |
| `sfx_upgrade.wav` | 升级音效 |
| `sfx_coin.wav` | 金币音效 |
| `sfx_wave_start.wav` | 波次开始 |
