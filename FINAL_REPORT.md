## 🚀 使用步骤

### 1. 导入Unity
```
1. 创建Unity 2D项目
2. 复制Assets/Scripts到项目
3. 等待编译完成
```

### 2. 生成占位资源
```
菜单: Tools → Create Placeholder Assets
生成精灵和材质
```

### 3. 快速设置场景
```
菜单: Tools → Quick Scene Setup
一键创建场景和预制体
```

### 4. 运行测试
```
按Play按钮测试游戏
```
