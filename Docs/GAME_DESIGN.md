# 🧟 僵尸防线 - 游戏设计文档

## 1. 游戏概述

### 1.1 核心定位
- **类型**: 塔防射击 + Roguelike
- **平台**: 微信小游戏、移动端
- **目标用户**: 18-35岁，喜欢休闲策略游戏的玩家
- **游戏时长**: 单局10-30分钟

### 1.2 一句话描述
> 建造防御塔，抵御无尽僵尸潮，每波选择技能Build，挑战生存极限！

## 2. 核心玩法

### 2.1 基础循环
```
开始波次 → 生成敌人 → 自动战斗 → 消灭所有敌人 → 技能选择 → 下一波
```

### 2.2 防御塔系统
- **自动攻击**: 范围内自动锁定并攻击敌人
- **手动控制**: 玩家可拖动调整防御塔位置
- **升级路线**: 攻击力/射程/射速/特殊效果

### 2.3 Roguelike技能系统
每波结束后三选一：

| 技能类型 | 效果 | 稀有度 |
|---------|------|--------|
| 火力强化 | 攻击力+20% | 普通 |
| 穿甲弹 | 子弹穿透2个敌人 | 稀有 |
| 爆裂弹 | 范围伤害 | 稀有 |
| 双重射击 | 每次2发子弹 | 史诗 |
| 冰冻弹 | 减速敌人 | 稀有 |

### 2.4 敌人系统

#### 普通僵尸
- 血量: 50
- 速度: 2
- 奖励: 10金币

#### 快速僵尸
- 血量: 30
- 速度: 4
- 奖励: 15金币

#### 坦克僵尸
- 血量: 200
- 速度: 1
- 奖励: 30金币

#### BOSS僵尸
- 每10波出现
- 血量: 1000+
- 特殊能力

## 3. 经济系统

### 3.1 收入来源
- 击杀敌人: 10-50金币
- 波次奖励: 100金币 × 波次数

### 3.2 消费点
- 技能刷新: 50金币
- 紧急修复: 100金币恢复1点基地生命

## 4. 变现设计

### 4.1 广告变现
- 复活广告: 死亡后观看广告继续
- 双倍奖励: 观看广告获得双倍金币
- 技能刷新: 免费刷新技能选择

### 4.2 内购
- 去广告: 18元永久去除
- 钻石: 用于购买皮肤
- 月卡: 30元，每日送钻石

### 4.3 皮肤系统
- 防御塔皮肤
- 子弹特效
- 场景主题

## 5. 社交功能

### 5.1 排行榜
- 周榜: 本周最高波次
- 总榜: 历史最高波次
- 好友榜: 微信好友排名

### 5.2 分享
- 分享复活
- 成绩分享
- 邀请好友奖励

## 6. 技术规格

### 6.1 性能目标
- 目标帧率: 60 FPS
- 同屏敌人: 最大50个
- 包体大小: < 20MB

### 6.2 分辨率适配
- 设计分辨率: 1080 x 1920
- 适配方案: UI Scale With Screen Size

## 8. 竞品分析

### 8.1 《向僵尸开炮》
- 优点: 爽快感强，技能丰富
- 缺点: 后期重复度高
- 差异化: 更强调策略Build

### 8.2 《遗弃之地》
- 优点: 中式恐怖题材独特
- 缺点: 变现较弱
- 差异化: 更完善的商业化
