# 🧟 僵尸防线 — 情感叙事设计

---

## 核心情感主题

> **"在末日中，什么让我们依然是人？"**

不是"打僵尸"，而是"守护剩下的人性"。

---

## 角色深度设计

### 玩家角色：无名哨兵

你不是英雄。你是黎明城第7防区一名普通哨兵。

开场时你在写一封信——给在"大寂静"中失散的女儿。你不知道她还活着。你只是每天写，写完了折成纸飞机，从哨塔上扔出去。这是你活着的唯一理由。

三天后，防线将迎来最大规模进攻。如果你活着，你会继续找她。如果死了……至少你写过了。

**这封信的内容会随着游戏进程逐渐揭示。**

---

### NPC档案：六个真实的人

#### 1. 老王 — 炊事班班长，59岁

```
"我以前是开面馆的。臊子面，一碗12块，加蛋加2块。"

老婆和儿子都在大寂静第一天没了。他申请上前线被拒绝——年纪太大。
于是他留在炊事班，每天给哨兵做饭。

他记得每个人爱吃什么。新兵来的第一天，他会问：
"吃面不？"

他偷偷在面里多加一个蛋。
他说这是他唯一能做的事。
他总是笑着说，"人老了，就剩这点手艺。"

——
第五关后，你回到食堂。老王的位子是空的。
灶台上还有一碗面，上面贴着一张纸条：
"小陈（那个不爱吃葱的），你的。老王。"

没人知道他去了哪。有人说他偷偷跟了远征队。
也有人说他只是走不动了，在废墟里坐下，没再起来。

食堂的墙上，他刻了一行字：
"这辈子做过面条，也守过城。够本。"
```

#### 2. 林晓 — 医疗兵，23岁

医学系大三，大寂静时在实习。她救的第一个人是她导师——导师被感染后求她开枪。

她没开。导师变成了寂灭体，杀了三个人，最后被哨兵打死。

她从此失眠。每天凌晨3点起来清点医疗物资，一个一个数——绷带23卷，止痛药7支，抗生素0。

她偷偷藏了一把手术刀。她说"下次我会用的"。但你看到她每次给伤员缝合时，手稳得像机器。

她的日记本第一页只写了一行字：**"如果再见到老师，我会告诉他：我开了。"**

第四关通关后，她拿出一个旧手机给你看。手机只有1%电量，里面存着她妈妈最后一条语音：
"晓晓，冰箱里有饺子。妈妈爱你。"

她每天开机听一次。1%的电量从来没掉过——她从来不敢听完。
```

#### 3. 赵铁 — 工程兵，31岁

沉默寡言。每天修墙、布铁丝网、埋地雷。从不和人聊天。

直到有一天，他在修墙时哼歌。是《茉莉花》。

有人问他唱什么。他没回答。

后来才知道，他在废墟里找到了妻子的项链。项链里有个迷你音乐盒，拧两圈就响。

他每天拧一次。他说"快没发条了"。所以他只拧半圈，这样能多用一天。

第三关的城墙是他修的。上面刻了一行小字：
"阿芳，城墙很厚。你放心。"

他的妻子叫阿芳。三年前在撤离时走散。他觉得她在城墙的另一边。即使所有的地图都说那边只有废墟。
```

#### 4. 小刀 — 侦察兵，17岁

最小的哨兵。她谎报了年龄——实际只有15。被识破后，指挥官沉默了很久，说："那就学快点。"

她速度最快，反应最敏锐。但每次侦察回来，她会一个人蹲在角落里。

不是害怕。是在背名字。

每个在她面前死去的哨兵，她都会记住名字，记在手臂上。密密麻麻，从手腕到手肘。

"我怕没人记得他们。"她说。

有人问她怕不怕死。她想了想："怕。但我更怕我死了，没人记得我手臂上的名字。"
```

#### 5. 周教授 — 前生物学家，52岁

大寂静前是国内顶尖基因学家。现在他在一个废弃实验室里研究寂灭体的基因结构。

他老婆在感染初期主动把自己隔离在实验室里，让他"解剖自己找到抗体"。

他没找到抗体。他花了三年。

但他发现了一个秘密——寂灭体不是随机攻击的。它们在"采集"。采集人类DNA中特定的序列。那些被"采集"的人，还活着。

他老婆的DNA特征就在采集清单的前排。

他现在的研究目标变了——不是消灭它们，是找到那些被采集的人的下落。

在他的实验室墙上，写着一行红色的字：
**"她还活着。我知道。"**
```

#### 6. 指挥官·赵 — 第7防区最高长官，44岁

她从来没有笑过。不是因为严肃。

第一波进攻时，她下令关闭内城门，把一支12人小队的退路断了——包括她的丈夫。

她用12条命换来了2000平民撤退的时间。

每天晚上10点整，她会把对讲机调到旧频道，说一声"收队"。
从来没人应答。

她的桌子上压着一张照片——她和一个男人的合影。照片背面写着：
"你说过会回来。"

"所以我一直等。"
```

## 情感触发点设计

### 关卡内触发

| 关卡 | 触发条件 | 内容 |
|:---:|------|------|
| 1-3 | 首次放置箭塔 | 小刀说："那个位置我以前站过。" |
| 2-5 | Boss HP<50% | 周教授通过对讲机："等一下——它的基因序列里有阿梅的片段……" |
| 3-2 | 城墙被破坏 | 赵铁不说话，但修复速度快了两倍 |
| 4-3 | 侦察兵小刀阵亡（剧情杀，可选避免） | 解锁她的隐藏任务线 |
| 5-1 | 玩家首次战败 | 所有人发来语音，只有老王说："回来吃面。" |

### 战役间触发

```
每个章节结束后，回到黎明城时：

1. 老王的面馆 — 一段对话
2. 林晓的医疗站 — 一封未寄出的信
3. 赵铁的城墙 — 一行新刻的字
4. 小刀的记事本 — 新增的名字
5. 周教授的实验室 — 新的发现
6. 指挥官的指挥室 — 对讲机里的沉默
```

### 核心情感场景（详细剧本）

#### 场景A：第五关后·老王的面

```
【场景：食堂，夜里】

（玩家走进食堂。灯还亮着，灶台上放着一碗面和一封信。）

[信的内容——手写体，字迹潦草但用力]

"哨兵同志：

我要去找一个人。我听说她在北边的难民点。

我怕再不去就来不及了。你知道的，我这把年纪。

走之前给你下了碗面。油泼的，你说过辣一点好吃。

冰箱里还有三天的食材。赵铁会做。别让他放太多盐。

我儿子小时候也爱吃我做的面。
后来没了。
后来来了你们。
我就又有孩子了。

所以我不走，就是在等死。
走了，说不定还能再见一个。

谢谢你。谢谢你们。
谢谢你们让我在最后这些年，还是个爸爸。

—— 老王"

（碗里的面还冒着热气。）

（玩家可以选择：吃完 / 留到明天。）

如果吃了：获得永久Buff"老王的味道"——所有哨兵HP回复速度+10%
如果留着：第二天小刀会来问你"你也没吃啊？"解锁隐藏对话
```

#### 场景B：林晓的日记

```
【场景：医疗站，深夜】

（每次章节通关后，林晓的日记本会更新一页。）

第一页（已有）：
"如果再见到老师，我会告诉他：我开了。"

第二页（第二章后）：
"今天救了一个和妈妈差不多年纪的伤员。
她醒来问我在哪，我说黎明城。
她愣了一下，说：'那是个好名字。'
她说她女儿也叫晓晓。
我没说话。我假装在整理药品。"

第三页（第三章后）：
"那个伤员走了。
临走前她给了我一个橘子。真的橘子。
三年没吃过了。
我分了三瓣给小刀。剩下的我放在枕头边。
橘子的味道让人想哭。但我没哭。"

第四页（第四章后）：
"今天从废墟里找到了一个手机。
不是我妈那一部。
但电量也是1%。
我把它和小刀背名字的记号笔放在一起。
有些事，1%的电量就够了。"

第五页（终章后）：
"老师：

我开了。
不是那把手术刀。是心里那把。

我救的人比我不敢救的人多了。
所以这一刀，算我赢了。

你教我的最后一课，我补考及格了。

—— 晓"
```

#### 场景C：赵铁的城墙

```
【场景：城墙，黄昏】

（赵铁每天黄昏会来城墙坐一会儿。如果你也来，他会偶尔说话。）

第一天：
"（沉默）"

第三天：
"这堵墙修了三次了。每次修好，第二天又破了。"
"但我还是修。"
"因为墙那边……"

第七天：
"阿芳喜欢茉莉花。她名字里就有个芳字。"
"这首歌是她教我的。我唱得不好听，她每次都笑着捂耳朵。"
"但我还是唱。"
（播放《茉莉花》片段，跑调版）

第十五天（如果之前每次黄昏都来了）：
"你天天来，不巡逻吗？"

玩家可以选择：
A: "你天天修，不累吗？"
B: "这堵墙，会破吗？"
C: "（沉默，坐下）"

选择C后，赵铁会给你一样东西——一个旧音乐盒。
他说："拧半圈。那个她留给我的。这个给你。"
"等城墙不会破了，你还给我。"
```

## 环境叙事元素

### 可收集物品（散布在各关卡）

| 关卡 | 物品 | 描述 |
|:---:|------|------|
| 1-2 | 旧照片 | 一家三口的合影，背面写着"2021年春节，武汉" |
| 2-3 | 日记残页 | "Day 47：水快没了。我知道她在骗我——每次她都说她喝过了。" |
| 3-1 | 玩具熊 | 一只缝了补丁的泰迪熊。上面缝着"给豆豆，爸爸" |
| 4-2 | 录音笔 | 最后一条录音：一个孩子的声音"我希望明天能去游乐园" |
| 5-3 | 信纸 | "致看到这封信的人：我叫李明。这是我的故事——"然后纸张被撕掉了 |

### 城市废墟中的细节

- 一个还没倒闭的奶茶店招牌，"第二杯半价"的灯还在闪
- 斑马线上停着一辆婴儿车，空的
- 广告牌上明星的笑脸被弹孔打穿
- 消防栓上贴着一张纸条："还有人在吗？我们在XX区。有食物。——幸存者"

## 叙事节奏设计

### 整体情感曲线

```
关卡1-2:   平静→紧张  (新手期，建立关系)
关卡3:     紧张→悲伤  (第一次伤亡)
关卡4:     悲伤→愤怒  (发现真相)
关卡5:     愤怒→决心  (Boss战，阶段胜利)

间章:      回顾→温暖  (回城，NPC对话)

关卡6-7:   温暖→不安  (远征开始)
关卡8:     不安→恐惧  (敌人升级)
关卡9:     恐惧→绝望  (陷入包围)
关卡10:    绝望→希望  (Boss战，揭露线索)

间章:      希望→迷惑  (新信息改变认知)

关卡11-13: 迷惑→怀疑  (进入遗迹)
关卡14:    怀疑→震惊  (真相揭露)  
关卡15:    震惊→觉醒  (人类宿命)

间章:      觉醒→选择  (三条路线)

关卡16-20: 取决于玩家选择——复仇？理解？共存？
```

---

**叙事原则**：
1. 不写英雄，写人
2. 不用旁白说"他很难过"，用细节——一碗面、半圈发条、1%的电量
3. 死亡可以发生，但要有重量。每一个死去的人都要被记住
4. 幽默和温暖要穿插——末日里人们依然会笑
5. 玩家的每个选择，都有人为此付出代价
