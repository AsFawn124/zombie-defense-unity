## 世界观：黎明纪元

### 时代背景

**2147年，新纪元3年。**

三年前，"大寂静"降临——全球量子通信网络同时收到一段来自深空的信号。信号解码后是一组基因序列，被各国实验室争相合成。三个月后，第一批合成生物失控，全球陷入"第一次生化战争"。人类文明在短短6个月内被推到灭绝边缘。

幸存者在最后堡垒城市**「黎明城」**中重建文明。他们发现，那些被感染的生物——**"寂灭体"**——并不只是无脑的怪物。它们是被某种更高等的宇宙意志控制的生物兵器，目的是"净化"碳基文明。

而人类，是实验品。

### 核心冲突

- **人类残存势力** vs **寂灭体潮**：地表97%已被污染，黎明城是人类最后的火种
- **净化议会** vs **融合派**：城内分裂——一方要求彻底消灭所有感染，一方认为可以与低级别寂灭体融合获得力量
- **远古真相**：信号来源并非外星文明，而是人类自己在100年前发射的深空探测器发回的……来自未来的警告

### 三大势力

| 势力 | 理念 | 代表英雄 |
|------|------|---------|
| 🔴 **黎明军团** | 武力消灭所有寂灭体 | 指挥官·铁壁 |
| 🔵 **融合先驱** | 反向利用寂灭体基因 | 博士·千面 |
| 🟣 **守望者** | 寻找信号源头，终结循环 | 先知·回声 |

## 新玩法机制

### 1. 昼夜循环系统 ⏳

```
白天（6回合）  →  黄昏（2回合）  →  黑夜（4回合）  →  黎明（2回合）  →  循环

白天：视野100%，寂灭体速度正常
黄昏：视野80%，寂灭体攻速+20%（狂暴状态）
黑夜：视野50%，寂灭体伤害+50%，但掉落双倍金币
黎明：视野恢复中，寂灭体变慢但数量激增
```

### 2. 天气系统

| 天气 | 出现概率 | 效果 |
|------|:---:|------|
| 晴天 | 40% | 正常状态 |
| 暴雨 | 20% | 电塔伤害+30%，火塔伤害-50%，移动速度-20% |
| 沙尘暴 | 15% | 视野-30%，飞行敌人无法出现 |
| 暴风雪 | 10% | 所有寂灭体速度-30%，但HP+50% |
| 辐射风暴 | 10% | 寂灭体持续变异，随机获得强化 |
| 极光夜 🌟 | 5% | 稀有事件！所有塔攻速+50%，掉落翻倍 |

### 3. 英雄系统

每位英雄拥有**3个专属技能**，战斗中手动释放：

| 英雄 | 技能1 | 技能2 | 技能3（终极技） |
|------|------|------|------|
| 铁壁·指挥官 | 空袭：对区域造成300%伤害 | 鼓舞：全塔攻速+50%持续10秒 | **钢铁风暴**：全屏500%伤害 |
| 千面·博士 | 基因炸弹：范围毒伤 | 融合光环：塔射程+30% | **基因突变**：随机塔升级+3 |
| 回声·先知 | 时间减速：敌人减速80%/5秒 | 预知：下波敌人信息提前显示 | **时间回溯**：回到10秒前状态 |
| 影刃·猎手 | 暗杀：秒杀血量<20%的敌人 | 陷阱：地面陷阱持续伤害 | **暗影风暴**：召唤暗影分身 |

### 4. 塔防进化系统

同一座塔击杀足够敌人后可进化：

```
箭塔(50kill) → 连弩塔(100kill) → 暴风弩炮(200kill)
                  ├→ 毒箭塔(100kill) → 剧毒箭阵
                  └→ 穿甲箭塔(100kill) → 破魔弩

炮塔(50kill) → 榴弹炮(100kill) → 末日巨炮(200kill)
                  ├→ 燃烧炮(100kill) → 地狱火
                  └→ 冰冻炮(100kill) → 极寒风暴
```

### 5. 战术支援系统

每波可在3个随机支援中选择1个：

| 支援类型 | 示例 |
|---------|------|
| 🎯 火力支援 | 空袭、导弹齐射、轨道炮 |
| 🛡️ 防御增援 | 临时护盾、维修机器人、城墙加固 |
| 💰 经济支援 | 金币补给、免费升级、利息翻倍 |
| ⚡ 特殊支援 | EMP炸弹（机械敌人即死）、信号弹（下波延迟）、诱饵 |

### 6. 无尽深渊模式 🌊

打通第4关后解锁，无限波次+排行榜：

- 每10波出现随机Boss
- 每5波获得一次额外选牌/升级机会
- 每20波切换环境
- 全服竞速排行榜（最高波数/最少时间）
- 每日首通深渊奖励

## 剧情写作指南

### 基调
末世但不绝望——人类在绝境中展现的勇气和智慧是主线。

### 叙事手法
1. **关卡前CG**：30秒短动画交代剧情（2D静态图片+配音即可）
2. **关卡内对话**：英雄在关键波次触发语音
3. **收集品叙事**：每个关卡隐藏1-2个"旧世界遗物"，阅读可了解世界细节
4. **角色日记**：打完章节解锁该英雄的背景故事

### 关键剧情转折点
- ⚡ 第5关后：发现寂灭体有"指挥者"——它们是被控制的
- ⚡ 第10关后：石油平台数据揭示——这不是第一次
- ⚡ 第15关后：真相大白——人类是第7代文明，前6代全部灭绝
- ⚡ 第20关后：打破循环的关键——不是消灭，而是理解
