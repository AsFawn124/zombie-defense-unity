## 第一季详细关卡设计

### 关卡1: 外墙哨站
```
环境: 荒漠 / 白天
地图大小: 800x600
路径数: 1条主路
可建塔位: 12个

Wave 1   x5 爬行者 (新手教程)
Wave 2   x8 爬行者 (解锁箭塔)
Wave 3   x10 爬行者 (解锁炮塔)
Wave 4   x5 爬行者 + x3 变异犬 (解锁冰塔)
Wave 5   x12 爬行者 + x5 变异犬
Wave 6   x8 变异犬 + x2 跳跃者 (解锁电塔)
Wave 7   x15 爬行者 (高速波，练习快节奏)
Wave 8   x10 变异犬 + x3 跳跃者 (强度测试)
最终波  x20 敌人混合 (Boss战前奏)
```

### 关卡5: 母巢之战 🔴 BOSS关

```
环境: 地下洞穴 / 黑暗
特殊机制: 
- 每3波母巢释放一波小怪
- 母巢周期性回血(需要高DPS打断)

Phase 1 (HP 100%-70%) - 母巢孵化期
  每15秒: 生成x3 变异爬行者
  攻击方式: 触手打击(50伤害，前方直线)

Phase 2 (HP 70%-30%) - 狂暴孵化期  
  每10秒: 生成x5 变异爬行者 + x1 精英爬行者
  攻击方式: 触手横扫(80伤害，扇形) + 酸液喷射(范围DOT)

Phase 3 (HP 30%-0%) - 垂死挣扎期
  召唤全部储备: x10 变异爬行者 + x2 精英
  攻击方式: 全屏地震(所有塔暂停攻击3秒)
  
奖励: 
  首通: 母巢核心(饰品) + 500金币 + 50钻石
  重复: 100金币 + 母巢材料(用于塔进化)
```

## 塔防进化路线

### 箭塔系
```
箭塔 Lv1
├→ Lv2 连弩 (+20%攻速, +10%伤害)
│  ├→ Lv3 暴风弩炮 (穿透3目标)
│  └→ Lv3 追踪弩 (自动瞄准最弱敌人)
├→ Lv2 毒箭塔 (中毒DOT 5秒)
│  └→ Lv3 剧毒箭阵 (范围中毒)
└→ Lv2 破甲箭塔 (无视30%护甲)
   └→ Lv3 破魔弩 (对Boss伤害+100%)
```
