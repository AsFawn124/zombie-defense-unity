# 僵尸防线 - Unity塔防射击游戏

## 项目介绍
一个完整的Unity塔防射击游戏，融合Roguelike元素。

## 功能特性
- 7种不同类型的敌人
- 12个Roguelike技能
- 完整的UI系统
- 微信小游戏适配
- 关卡编辑器

## 技术栈
- Unity 2022.3 LTS
- C#
- 微信小游戏SDK

## 项目结构
```
Assets/
├── Scripts/        # 游戏脚本
├── Resources/      # 游戏资源
├── Scenes/         # 游戏场景
└── Prefabs/        # 预制体

Docs/               # 项目文档
```

## 快速开始
1. 克隆项目
2. 在Unity中打开
3. 运行游戏
